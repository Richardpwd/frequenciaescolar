# Avance - Apoio Escolar

Sistema completo para controle de frequência escolar, com autenticação, cadastro de alunos, salas, calendário e usuários. Backend em Node.js/Express/SQLite, frontend responsivo em HTML/CSS/JS.

## Como rodar

1. Instale as dependências no backend:
   ```
   cd backend
   npm install
   ```
2. Inicie o backend:
   ```
   npm start
   ```
3. Acesse http://localhost:3000 no navegador.

---

- Login padrão: cadastre um novo usuário.
- Layout azul/branco, responsivo, com logo fictícia.
- Estrutura MVC (routes, controllers, models, views).
